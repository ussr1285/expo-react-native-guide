## 2. Expo 개발 가이드
* **1-1. Expo 개발 관련 참고 사이트**  
expo는 expo에서 제공하는 패키지만 사용가능합니다.
리액트 네이티브에서 사용할 수 있는 여러 패키지들은 expo에서는 사용할 수 없다고 보면 됩니다.
그러므로 expo에서 제공한 공식문서에 없는 패키지는 쓸 수 없다고 보시면 됩니다.

expo 공식 문서 : https://docs.expo.io/
